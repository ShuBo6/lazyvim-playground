# LazyVim 快速参考卡片

## 最常用快捷键（前20个）

### 必记快捷键
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Esc` | 返回 Normal 模式 | 最重要！ |
| `i` | 进入插入模式 | 在光标前插入 |
| `a` | 进入插入模式 | 在光标后插入 |
| `:w` | 保存文件 | Write |
| `:q` | 退出 | Quit |
| `:wq` | 保存并退出 | Write & Quit |
| `u` | 撤销 | Undo |
| `Ctrl+r` | 重做 | Redo |

### 文件操作
| 快捷键 | 功能 |
|--------|------|
| `<Space>e` | 打开/关闭文件树 |
| `<Space>ff` | 查找文件 |
| `<Space>fg` | 全局搜索文本 |
| `<Space>fb` | 查找已打开的文件 |
| `<Space>fr` | 最近打开的文件 |

### 代码导航
| 快捷键 | 功能 |
|--------|------|
| `gd` | 跳转到定义 |
| `gr` | 查找引用 |
| `K` | 显示文档 |
| `<Space>ca` | 代码操作 |
| `<Space>cr` | 重命名符号 |

### 窗口操作
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+h/j/k/l` | 移动到左/下/上/右窗口 |
| `<Space>-` | 水平分割窗口 |
| `<Space>\|` | 垂直分割窗口 |

## 移动速查表

### 基础移动
```
h ←    j ↓    k ↑    l →

w - 下一个单词      b - 上一个单词
0 - 行首           $ - 行尾
gg - 文件开头      G - 文件结尾

Ctrl+u - 上翻半页   Ctrl+d - 下翻半页
```

### 快速跳转
```
f{char} - 跳到字符    F{char} - 反向跳
/{text} - 搜索        n - 下一个匹配
* - 搜索当前单词
```

## 编辑速查表

### 基本编辑
```
进入插入模式：
i - 光标前    a - 光标后

o - 下方新行  O - 上方新行

删除：
x - 删除字符   dd - 删除行
```

```
v - 字符选择

Ctrl+v - 块选择

选择后：
d - 删除
y - 复制
```

## 紧急情况

### 我不知道怎么退出 Vim！
```
按 Esc 键几次，然后输入：
:q!   (不保存退出)
:wq   (保存并退出)
```

### 我改错了怎么办？
```
按 Esc 回到 Normal 模式
按 u 撤销
按多次 u 可以一直撤销
```

### 我不小心录制宏了
```
按 q 停止录制
```

### 屏幕显示很奇怪
```
按 Esc 回到 Normal 模式
输入 :e! 重新加载文件
或者 Ctrl+l 刷新屏幕
```

## 配置 Golang 环境步骤

### 1. 打开 LazyVim
```bash
nvim
```

### 2. 打开 Lazy 管理器
```
按 <Space>l
```

### 3. 进入 Extras
```
按 x
```

### 4. 启用 Golang 支持
```
找到 lang.go
按 <Space> 或 Enter 启用
```

### 5. 安装依赖
```
按 I 安装插件
等待安装完成
```

### 6. 安装 Golang 工具
```bash
go install golang.org/x/tools/gopls@latest
go install github.com/go-delve/delve/cmd/dlv@latest
```

### 7. 重启 Neovim
```
:q
nvim your_file.go
```

## 常用命令速记

### Ex 命令
```
:w          保存
:q          退出
:wq 或 :x   保存并退出
:q!         强制退出（不保存）
:e file     编辑文件
:sp         水平分割
:vsp        垂直分割
:term       打开终端
:Mason      管理 LSP 服务器
:Lazy       管理插件
:help       帮助文档
```

### 实用组合
```
ggVG        全选
:%y+        复制全文到剪贴板
:%d         删除全文
:sort       排序选中行
:!{cmd}     执行外部命令
```

## 提示与技巧

1. **不要一直停留在 Insert 模式**
   - 养成编辑完就按 Esc 的习惯
   - Normal 模式才是 Vim 的主场

2. **多用相对行号跳转**
   - 看到目标在下方 5 行：`5j`
   - 删除下方 3 行：`3dd`

3. **善用 `.` 命令重复操作**
   - 执行一次操作后，用 `.` 快速重复

4. **使用 `*` 搜索当前单词**
   - 光标移到变量名上，按 `*` 查找所有出现位置

5. **`Ctrl+o` 和 `Ctrl+i` 在跳转历史中前后移动**
   - 跳转到定义后想回来：`Ctrl+o`

6. **用 `zz` 让当前行居中**
   - 方便查看上下文

7. **使用 `ciw` 快速修改单词**
   - Change Inner Word - 删除当前单词并进入插入模式
